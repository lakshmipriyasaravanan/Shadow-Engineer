# Security Policy

## Supported Versions

We currently provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within Shadow Engineer, please send an e-mail to security@shadowengineer.com. All security vulnerabilities will be promptly addressed.

Please **DO NOT** report security vulnerabilities via public GitHub issues.

### Response timeline
- We will acknowledge receipt of your vulnerability report within 24 hours.
- We will provide a timeline for a fix within 72 hours.
