# Roadmap v2.0

As we celebrate the v1.0 launch, here is the vision for Shadow Engineer v2.0:

## Q4 2026: The Agentic Era
- **Autonomous Issue Resolvers**: Shadow Engineer will not just review PRs, it will read GitHub Issues, branch out, write the fix, and open the PR entirely autonomously.
- **Multi-Agent Debates**: Introducing specialized sub-agents (e.g., `Security_Agent`, `Performance_Agent`) that debate architectural decisions before generating code.

## Q1 2027: IDE Integration
- **VS Code & IntelliJ Extensions**: Bring the Developer Copilot directly into the IDE.
- **Real-time Context Tracking**: The AI will understand what file you are currently looking at and proactively suggest optimizations.

## Q2 2027: Multi-Cloud & Enterprise Deployments
- **Azure & GCP Support**: Expand Terraform modules beyond AWS to support Azure AKS and Google GKE.
- **SSO/SAML Integrations**: Support Okta and Azure AD for massive enterprise adoptions.
- **SOC2 Compliance Reporting**: Automated generation of compliance artifacts.
